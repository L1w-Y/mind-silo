## 为什么要搭建个人网站

一直想有一个属于自己的空间，可以记录技术笔记、收藏好文、沉淀想法。

市面上有很多博客平台，但我更希望拥有完全的控制权——从设计到数据。

## 技术选型

这个网站使用了以下技术栈：

- **Next.js** — React 生态的全栈框架
- **Tailwind CSS** — 原子化 CSS 工具
- **MDX** — 在 Markdown 中使用 React 组件

## 一段示例代码

const greeting = (name: string): string => {
  return `Hello, ${name}! Welcome to my blog.`;

console.log(greeting("World"));
