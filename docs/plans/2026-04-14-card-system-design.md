# 卡片系统 + 纸质质感背景 设计方案

> 日期：2026-04-14
> 状态：已确认

## 概述

将博客、日常、收藏、想法 4 个页面改造为 Craft/Notion 风格的彩色卡片网格布局，每个页面有独立的圆角纸质质感背景板。

## 1. 纸质质感

所有页面背景叠加 CSS SVG noise 纹理，模拟纸张颗粒感：
- 使用 `<svg>` feTurbulence 生成噪点
- 半透明叠加在背景色上
- 噪点密度适中，不干扰阅读

## 2. 页面背景板

每个页面有独立的圆角背景（与首页天空背景同级）：

| 页面 | 背景色 | 渐变方向 | 感觉 |
|------|--------|---------|------|
| 首页 | `#a8d4f0` → `#e8f4fc` | 上→下 | 天空蓝（已有） |
| 博客 | `#f0e6d6` → `#f7f2eb` | 上→下 | 暖米/羊皮纸 |
| 日常 | `#e4daf0` → `#f2eef7` | 上→下 | 薰衣草紫 |
| 收藏 | `#d4edda` → `#eef6f1` | 上→下 | 薄荷绿 |
| 想法 | `#f5edd4` → `#faf6ed` | 上→下 | 奶油黄 |

所有背景：`border-radius: 24px` + 噪点纹理叠加

## 3. 卡片系统

### 色板（8色）

| 名称 | 边框色 | 背景色 | 匹配标签 |
|------|--------|--------|---------|
| red | `#e8846b` | `#fdf2ef` | React, 前端 |
| orange | `#d9903e` | `#fdf5ed` | 工具, CLI |
| yellow | `#c4a83b` | `#fbf8ef` | 笔记, 日常 |
| green | `#5ba67c` | `#eef6f1` | 开源, Go |
| teal | `#4da8a0` | `#eef7f6` | 文档, 教程 |
| blue | `#5b8fc9` | `#eff4fb` | TypeScript, Next.js |
| purple | `#9b7ec8` | `#f4f0fa` | AI, 设计 |
| pink | `#d47da0` | `#fbf0f4` | 生活, 收藏 |

### 颜色分配逻辑
1. 数据库有 `color` 字段 → 使用手动颜色
2. 没有 `color` → 按第一个标签名 hash 自动映射到 8 色之一
3. 没有标签 → 使用默认蓝色

### 卡片样式
- 圆角: 12px (标准) / 16px (大卡片)
- 边框: 左侧 3px 彩色边条 + 整体 1px rgba(0,0,0,0.08)
- 背景: 对应色板的淡色背景
- 阴影: Notion 4层阴影 `rgba(0,0,0,0.04) 0px 4px 18px, ...`
- 文字: `rgba(0,0,0,0.95)` 暖黑

### 卡片尺寸
- full: col-span-12 (整行)
- half: col-span-6 (半行)  
- third: col-span-4 (三分之一)
- 数据库字段: `card_width` TEXT DEFAULT 'half'

## 4. 各页面布局

### 博客页
- 文章卡片网格，默认 half 宽
- 卡片内容：标题 + 摘要 + 日期 + 标签 + 阅读时间
- 点击打开阅读面板（保持现有）

### 日常页
- 日记卡片网格，按日期分组
- 卡片宽度可调（full/half/third）
- 保持拖拽排序功能

### 收藏页
- 收藏卡片网格，默认 third 宽
- 卡片内容：favicon + 标题 + 摘要 + 标签
- 标签筛选保持

### 想法页
- 想法卡片网格，默认 third 宽
- 像便签散落，按日期分组
- 每张卡片颜色不同（由标签决定）

## 5. 数据库变更

给 4 张表各加两个字段：
- `color` TEXT DEFAULT NULL — 手动颜色覆盖
- `card_width` TEXT DEFAULT 'half' — 卡片宽度

## 6. 实现步骤

1. 添加噪点纹理 CSS + 各页面背景板样式
2. 创建通用 ContentCard 组件（彩色边条+背景+尺寸）
3. 创建颜色映射工具函数
4. 数据库加字段
5. 改造博客页
6. 改造收藏页
7. 改造想法页
8. 改造日常页
